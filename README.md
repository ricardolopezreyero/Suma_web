# Coahuila — Tierra de Gigantes

Sitio estático de promoción de inversión para Coahuila, listo para GitHub Pages.

## Publicación

1. Sube la rama `main` al repositorio.
2. En GitHub: **Settings → Pages → Build and deployment**.
3. Selecciona **Deploy from a branch**, `main` y `/ (root)`.
4. La URL pública será `https://ricardolopezreyero.github.io/Suma_web/`.

## Notas

- El sitio está en español; el selector de idioma ya está preparado como interfaz para futuras traducciones.
- Las fotografías se cargan desde Unsplash como placeholders visuales. Antes del lanzamiento institucional, reemplázalas por fotografía propia o activos con licencia y autorización documentada.
- Las métricas visibles apuntan a la ficha pública de la Secretaría de Economía; validar cifras y fechas antes de cada publicación.
